## üéØ Common Scenarios

### Building a Hero Section
1. Use `AnimatedBackground` for orbs
2. Use `staggerContainer` for content
3. Add `AnimatedButton` for CTA
4. Reference: [ANIMATIONS-QUICK-START.md](./ANIMATIONS-QUICK-START.md#-complete-example)

### Creating a Card Grid
1. Use `StaggerList` wrapper
2. Wrap each card in `StaggerItem`
3. Use `AnimatedCard` for hover effects
4. Reference: [client/ANIMATIONS-GUIDE.md](./client/ANIMATIONS-GUIDE.md#card-grid-with-stagger)

### Adding Loading States
1. Use `LoadingSpinner` for full page
2. Use `LoadingDots` for inline
3. Use `LoadingPulse` for decorative
4. Reference: [ANIMATIONS-QUICK-START.md](./ANIMATIONS-QUICK-START.md#6-show-loading-state)

### Animating Statistics
1. Use `AnimatedCounter` for numbers
2. Add `ScrollReveal` wrapper
3. Stagger multiple counters
4. Reference: [client/ANIMATIONS-GUIDE.md](./client/ANIMATIONS-GUIDE.md#stats-counter-section)
