1. INSTALL GIT (if not installed)
   → https://git-scm.com/download/win

2. CREATE ACCOUNTS
   → GitHub: https://github.com/signup
   → Railway: https://railway.app (use GitHub login)

3. DEPLOY BACKEND
   a) Create repo on GitHub: 'mre-backend'
   b) Run these commands in server folder:
   
      git init
      git add .
      git commit -m \"Initial commit\"
      git remote add origin https://github.com/YOUR_USERNAME/mre-backend.git
      git push -u origin main

   c) Deploy on Railway:
      - New Project → Deploy from GitHub
      - Select 'mre-backend'
      - Add environment variables from .env file
      - Get your Railway URL

4. CONFIGURE FRONTEND
   → Come back with your Railway URL
   → I'll update your frontend to connect to it
